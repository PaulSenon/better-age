---
title: "@initBetterAge(...)"
description: Root decorator that configures which payload and launcher the plugin should use.
---

## Purpose

`@initBetterAge(...)` initializes the plugin once for the current varlock process.

## Minimal form

```txt
# @initBetterAge(path=.env.enc)
```

## Optional launcher override

```txt
# @initBetterAge(path=.env.enc, command="pnpm exec bage")
```

## Required fields

- `path` is required
- `command` is optional

## Validation rules

- `path` must be a non-empty string
- `command`, when provided, must be a non-empty string
- v0 supports only one init config per process

If the plugin is initialized again with a different config, it throws instead of silently switching.
