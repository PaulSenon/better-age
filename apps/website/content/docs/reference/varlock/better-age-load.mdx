---
title: "betterAgeLoad()"
description: Resolver function that returns env text by invoking the CLI load protocol.
---

## Purpose

`betterAgeLoad()` is the resolver function the varlock plugin exposes for env loading.

## Usage

```txt
# @setValuesBulk(betterAgeLoad(), format=env)
```

## Behavior

- takes no arguments
- fails if `@initBetterAge(...)` did not run first
- shells out to the CLI load contract
- returns env text on success

## Dependency

The function depends on:

```txt
bage load --protocol-version=1 <path>
```

That is why the CLI load protocol is the source of truth rather than the plugin inventing its own decrypt path.
