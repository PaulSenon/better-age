---
title: change-passphrase
description: Re-encrypt local key material under a new passphrase.
---

## Purpose

`change-passphrase` updates the passphrase used to protect local keys.

## Example

```sh
bage change-passphrase
```

## Behavior

- prompts for current passphrase
- prompts for the new passphrase twice
- updates passphrase protection for all local keys

## Output

On success the command reports:

```txt
updated passphrase for all local keys
```
