---
title: view
description: Open a payload in the secure viewer for human reading.
---

## Purpose

`view` is the human plaintext path.

It opens the secure in-process viewer instead of printing plaintext to `stdout`.

## Example

```sh
bage view .env.prod.enc
```

## Behavior

- accepts a payload path
- may guide path selection when possible
- is for humans, not machine piping

## Contract

`view` must not degrade into raw plaintext `stdout`.
