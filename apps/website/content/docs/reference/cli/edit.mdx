title: edit
description: Open a payload in your editor, then rewrite the encrypted payload from the edited env text.
---

## Purpose

`edit` is the main mutation path for changing secrets inside a payload.

## Example

```sh
bage edit .env.prod.enc
```

## Behavior

- can guide path selection in interactive use
- may require `update` first if the payload is stale
- depends on an available editor command
- uses temporary plaintext during the edit flow, then rewrites the encrypted payload

## Failure cases

Common failures include:

- payload needs update first
- editor command unavailable
- editor launch or exit failure
- newer unsupported payload version
