title: identities
description: Inspect local self identity, known identities, and retired keys.
---

## Purpose

`identities` prints a human-readable inspection of local identity state.

It includes:

- `Me`
- known identities
- retired local keys

## Example

```sh
bage identities
```

## Output shape

The command renders sections for:

- current self identity details
- known identities, including local aliases when present
- retired keys kept for historical decryptability

This is an inspection command, not a mutation command.
