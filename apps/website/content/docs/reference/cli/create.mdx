---
title: create
description: Create one visible encrypted payload file.
---

## Purpose

`create` creates a new encrypted payload file at the target path.

## Example

```sh
bage create .env.prod.enc
```

## Behavior

- path can be supplied directly
- guided resolution is possible when the path is omitted in interactive use
- setup must already exist

## Result

On success the command prints:

```txt
Created encrypted payload at <path>
```

The payload stays caller-owned and visible on disk.
