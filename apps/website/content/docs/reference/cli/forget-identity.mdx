---
title: forget-identity
description: Remove a saved known identity from local home state without touching payload access.
---

## Purpose

`forget-identity` is local-only cleanup.

It removes a known identity from local home state and does not mutate any payload.

## Example

```sh
bage forget-identity teammate#0123abcd
```

## Behavior

- can guide selection when the identity ref is omitted
- fails for ambiguous refs until you choose one exact identity
- refuses to forget your own self identity

## Important distinction

Use `forget-identity` for address-book cleanup.

Use `revoke` when you want to remove payload access.
