## Purpose

`interactive` opens the guided keyboard-first session.

Use it when you want the CLI to help complete intent through menus, prompts, and flow navigation instead of supplying every operand up front.

## Example

```sh
bage interactive
```

## Behavior

- depends on an interactive terminal
- can route into flows like edit, grant, or revoke
- is optimized for human usage, not headless automation

## When to use it

Use `interactive` when you want the CLI to lead. Use exact commands like `create`, `grant`, or `load` when you want a fully specified invocation.
