---
title: revoke
description: Remove one recipient from future rewritten versions of a payload.
---

## Purpose

`revoke` removes a recipient from future rewritten versions of a payload.

## Example

```sh
bage revoke .env.prod.enc teammate#0123abcd
```

## Behavior

- operates on the payload, not your local address book
- may require `update` first if the payload is stale
- fails distinctly if the payload version is newer than the CLI can open
- guided identity/path selection is possible in interactive use

## Important distinction

If you only want to remove a saved identity from local home state, that is a different operation: `forget-identity`.
