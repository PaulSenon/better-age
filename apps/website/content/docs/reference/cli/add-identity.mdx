---
title: add-identity
description: Save an external identity string as a known identity in local home state.
---

## Purpose

`add-identity` imports a shareable identity string into local home state.

This creates a known identity. It does not grant payload access by itself.

## Example

```sh
bage add-identity 'better-age://identity/v1/<base64url-json>'
```

## Behavior

- accepts a pasted identity string
- can run in guided mode when input is omitted
- rejects invalid identity strings
- rejects importing your own identity string

## Result

On success you get a saved known identity with display information and a handle you can use later with commands like `grant`.
